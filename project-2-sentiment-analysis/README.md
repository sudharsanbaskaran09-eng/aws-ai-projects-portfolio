# 💬 Project 2 — Sentiment Analysis Dashboard

## 📌 Overview
An AI-powered sentiment analysis app built on AWS that detects
whether text is Positive, Negative, Neutral or Mixed using
Amazon Comprehend.

## 🏗️ Architecture
User → API Gateway → Lambda → Comprehend → S3

## ⚙️ AWS Services Used
- Amazon Comprehend — AI sentiment analysis
- AWS Lambda — Serverless compute
- Amazon API Gateway — REST API
- Amazon S3 — Results storage
- AWS IAM — Security & permissions

## 💡 Real World Use Cases
- Customer review analysis
- Social media monitoring
- Product feedback analysis
- Support ticket prioritization

## 🚀 How to Run
1. Clone the repo
2. Configure AWS CLI with your credentials
3. Run sentiment_app.py locally
4. Or call the API endpoint with any text
